# Saidpiece Travel - Bhutan Experience 🏔️

A premium, modern travel landing page showcasing the beauty and wilderness of Bhutan. Built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**.

Bhutan Travel

## ✨ Features

- 🏗️ **Modern Architecture**: Component-based design using Next.js App Router.
- 🎨 **Rich Aesthetics**: High-quality imagery and clean typography inspired by Bhutanese landscapes.
- 📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.
- ⚡ **Performance First**: Fast landing page with optimized assets and server-side rendering.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📂 Project Structure

```text
src/
├── app/            # Next.js App Router (pages & layouts)
├── components/     # Reusable UI components
│   ├── Header.tsx

│   ├── FeaturedTrips.tsx
│   └── ...

public/
└── images/         # Optimized Bhutan photography
```

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🏗️ Building for Production

To create a production-ready bundle:

```bash
npm run build
```
